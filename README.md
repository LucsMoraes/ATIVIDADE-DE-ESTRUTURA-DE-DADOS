FUNÇÕES:

VETOR DE STRING:

STRUCT 

Tem como objetivo de Cadastrar 5 alunos e identificando como:

Nome (string)

Email (string)

Data de Nascimento (string)

5 Notas (float)

E as notas tem que ser somas e dá a média entre cada a aluno.
